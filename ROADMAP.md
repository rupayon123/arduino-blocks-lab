# Roadmap

Arduino Blocks Lab aims to become the easiest serious path from first circuit to real embedded development.

## What We Need To Beat

- LEGO SPIKE: friendly lessons, icon/word/text progression, delightful blocks, strong classroom flow.
- Arduino Cloud Editor: official board upload from browser with Arduino Cloud Agent.
- mBlock: Scratch-style blocks, text coding, extensions, and classroom adoption.
- Tinkercad Circuits: approachable simulation and wiring-first learning.
- Wokwi and similar tools: broad embedded simulation and shareable projects.

## Differentiators

- Real Arduino-first: generated Arduino C++ is the source of truth, not a hidden runtime.
- Any-board path: board targets use Arduino CLI FQBNs, so the platform can grow beyond starter boards.
- Sensor-pack system: components, blocks, libraries, wiring, examples, and lessons can ship as extension manifests.
- Learning ladder: icon-style missions, word blocks, generated C++, and Python/JavaScript previews.
- Public and remixable: no locked classroom ecosystem; contributors can add boards, sensors, and lessons.

## Near-Term Priorities

- Guided Mission Map: expand the new saved-progress mission path into a richer project map.
- Wiring Canvas: build on the new pin diagnostics with visual pin maps, conflict editing, and breadboard previews.
- Sensor Pack Gallery: grow the new URL/GitHub pack installer into a browsable public gallery.
- Better Upload Bridge: signed agent installers, first-run diagnostics, and clearer browser-to-localhost setup.
- Simulator Hooks: export Wokwi/Tinkercad-compatible project files where possible.

## Larger Bets

- “Explain My Circuit” assistant that checks wiring, code, libraries, and serial output together.
- Classroom mode with local-only rosters first, then optional sync.
- Community lesson marketplace with tests for generated code.
- Accessibility pass for keyboard block editing, screen-reader lesson flow, and high-contrast themes.
- Multi-language progression where Python and JavaScript become runnable for compatible boards.
